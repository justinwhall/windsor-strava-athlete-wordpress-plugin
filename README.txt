== Description ==

Windsor Strava athlete is a personal tool kit for Strava users. Display your profile, clubs, gear, stats, followers and more with shortcodes. 


== Installation ==

1. Upload `windsor-strava-athlete` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place short code `[windsor_strava_athlete]` in your templates. [Shortcode attributes](http://www.littlebot.io/docs/windsor-strava-athlete)
